Repositorio dedicado al aprendizaje y desarrollo de proyectos relacionados con **Lenguaje de Marcas**, realizados durante mi formación en **Desarrollo de Aplicaciones Multiplataforma (DAM)**.

Aquí se recopilan ejercicios, prácticas y proyectos que reflejan mi progreso desde los **conceptos más básicos de HTML y XML** hasta la creación de **páginas web más completas y estructuradas**.

## 📂 Contenido del repositorio

Este repositorio incluye diferentes proyectos organizados por nivel de dificultad y aprendizaje:

* **Fundamentos de HTML**

  * Estructura básica de una página web
  * Etiquetas esenciales
  * Listas, enlaces e imágenes

* **Maquetación web**

  * Uso de secciones y estructura semántica
  * Organización del contenido en páginas web

* **Formularios**

  * Creación de formularios HTML
  * Diferentes tipos de inputs
  * Validaciones básicas

* **XML**

  * Estructura de documentos XML
  * Uso de etiquetas y jerarquías
  * Validación y organización de datos

* **Proyectos prácticos**

  * Mini páginas web
  * Ejercicios de clase
  * Prácticas aplicadas
